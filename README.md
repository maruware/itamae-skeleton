# ItamaeSkeleton

Generating Itamae skeleton tool.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'itamae-skeleton'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install itamae-skeleton

## Usage

```bash
$ itamae-skeleton init
```
